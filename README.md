# Semantic-search-of-genes---Typer-integration
With this code you are able to search a biological process (e.g. "Photosynthesis") through your terminal, by typing python main.py &lt;query> (e.g. python main.py Photosynthesis) to get the 10 most relevant genes, or initiating isearch (python main.py isearch) for multiple searches and a customized number of results shown.
